Tweets - 流水账、发牢骚、想写啥就写啥

2018.04.14

1.
几天前我说我是个崇洋媚外的人, 但我已经很久没有这么形容自己了, 你可以看一下这篇《崇洋媚外》看看我为什么改变了想法
2.
在腾讯云开通了对象存储服务, 因为我这个博客是免费托管在 github 上的, github 在国内的访问速度胜任普通网页还没问题, 如果是大型图片(如: gif)就有点力不从心了, 况且没给钱太占用别人的带宽也不好
3.
随着这两天对腾讯云的熟悉, 由衷地感叹技术改变世界, 以前做什么都要自己动手搭建平台, 自己管理机房, 有了云, 所有资源现在都集中到一处(大公司)去了, 只要出钱, 有最好的硬件和人员帮助你实现你的点子

2018.04.13

1.
今天我在腾讯云开通了视频直播 LVB 服务, 使用 OBS 软件推流, 主要是用来跟人分享电影, 由于我的 teamviewer 收不到电脑内置声音, 所以一直想找一个可以设置密码的直播房间, 腾讯云用起来还不错, 资费也不贵, 不计推流上行流量, 一个 G 分发流量才几毛钱, 这意味着我分享一部电影给一个人看成本通常不会超过一块钱
2.
说一下我遇到的问题, 创建好频道后系统给了一个 rmtp 推流 URL, 我直接把它填到 OBS 了, 不过我在页面上怎么找也找不到 Stream Key 这个东西, 之前往 B站 推流是提供了 URL 和 Stream Key 的, 在网上搜索了一番原来腾讯给的 URL 已经包含了 Stream Key, 就是 URL 中 live/ 后面的那一串, 要自己手动拆分成推流地址和 Stream Key 两个部分

2018.04.10

1.
最近经常在 b站 主页看到五分钟左右的科技视频, 这些视频干货很少, 废话很多, 播音员的声音又不好听, 现实中好听的声音那么多, 不知道为啥这些做视频的就是喜欢播音腔呢
2.
智能手机音量键的设计问题: 现在的智能手机至少都有两个音频频道, 铃声和媒体, 铃声播放的是手机来电铃声和通知铃声的声音, 媒体则是播放手机音乐,视频, 游戏等声音, 这两个频道有独立的的音量条, 共享同一组音量键, 这本来不是问题, 但软件设计让人用起来非常不方便, 比如我现在身处安静的公共场所中, 正好想听个语音什么的, 这时候拿耳机肯定不方便, 所以我想先把媒体音量调小(2-4 格左右), 然后直接把手机放到耳朵附近来听, 可手机那一组音量键默认调节的是铃声音量, 而如果想调节媒体音量, 必须要手机正在播放媒体才行, 所以我想在播放这段语音前调小音量是不可能的, 我现在的解决方案是在 TIM 收藏里录一段静音的声音, 每当我要调节媒体音量时, 先播放这段安静的录音, 把音量调小, 再去播放我想听的语音, 我希望在按音量键的时候, 系统让我选择调节的是哪个音量, 就像这样:

箭头指向正在调节的音量, 在调节时如果用手指点击相应的音量条, 调节就会反馈到该音量上去. 对于耳机来说, 最好在插入的瞬间显示这个音量条, 以提醒用户注意音量大小.

3.
我今天把科普区分类属性从 audience 改为了 domain, 为 post 新增了 monthly_update 属性, 来告知读者该帖是会经常更新的, 取消内嵌 svg 代码, 使用 png 图片替换站点中的图标, 主页代码更好看了

2018.04.06

1.
百度首页的推送(和广告)近期消停了不少, 变得时有时没有了, 在搜索主页推送新闻有什么弊端呢? 它会在打开的瞬间分散人的注意力, 从而忘记了我想搜的题目, 很多时候一个搜索题目的产生是很突然的, 这种灵感很宝贵, 因为搜索这个题目也许能解决一个我疑惑了很久的问题, 同时也很容易忘掉, 百度推送起的标题太吸引人了, 我经常还没搜东西就点进去看了, 看着看着就忘了我本来想搜的题目. 注: 我讨论的是手机网页版
2.
一件超级后悔的事: 去年因为听黑金啊后摇啥的心情不好, 在 保存 我所有歌单后注销了网易云帐号, 我是一个崇洋媚外的人, 那时想着我还可以投靠 spotify 呀, 不比网易云音乐差吧, 后来转移歌单才发现很多歌 spotify 根本找不到, 国内的歌没有也就算了, 很多黑金属也搜不到, 而且听歌先听广告真是让我闻所未闻, 资料填的哪个地区就必须用那个地区的 ip 登录, 我后来用微信登的网易云音乐, 因为没有歌单, 已经有好长时间没怎么听歌了, 最近在想着把网易云歌单一首一首地恢复回来, 真的很麻烦啊, 这就是促使我写这个的原因了

2018.03.12

1.
我去年买了很多现在看来不必要的东西, 比如机械键盘, 这个东西用过之后才发觉没有想象中的那么神, 我用笔记本键盘打字也很舒服, 机械键盘的不便携性是没法用一点点手感提升来弥补的, 况且卖得又不便宜, 我准备抽时间找买家处理掉, 也考虑到我现在是本科最后一个学期了, 到时候太多东西也不容易搬
2.
第二个是我终于踩着大学的尾巴过了英语六级, 这次考了 495 分, 我背了半年单词, 平均每天 40-60 分钟, 六级成绩提高了 130 分, 这说明努力是可以带来成功的, 顺便吐槽一下查成绩的时候确实让人很紧张
3.
考完六级我感觉整个人有点松懈下来了, 背单词的积极性没有去年高, 希望我能战胜懒惰, 继续把背单词当成一个习惯
4.
这学期开始换用铅笔, 因为铅笔有记录时间长, 字迹易修改的特点, 这会无形中培养我做事的严谨性

2018.02.01

1.
完成了一直想开设的科普区, 我会时不时的在这里发布我认为有科普价值的小知识, 但考虑到网站的受众跟我有专业上的差异, 所以我划分了大众版和计算机科学版这两个分区, 大众版是针对所有人的, 目前讨论的话题有易混淆的词语, 计算机操作等, 而计算机科学版主要是介绍本专业的相关理论
2.
考虑到科普区的文章相对简短, 一个一个点进去看太过麻烦了, 我注意到阮一峰的网站首页有一个气泡式的评论展示版块, 于是把他网站的气泡功能搬了过来, 代码应该是开源的, 所以我可以使用, 现在把鼠标悬浮在一篇文章链接上面就有气泡显示文章内容, 当然内容太多还是要打开网页进行阅读的
