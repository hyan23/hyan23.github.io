Kindle升级和自定义字体

Kindle 最新版固件 5.9.6 已经支持使用自定义字体, 如果你的 Kindle 尚未收到推送(“更新您的 Kindle”选项为灰色), 可以按以下方法手动升级:

打开https://bookfere.com/{: target=”_blank” }, 下载适合你机型的最新固件, 下载完成后进行md5和sha1校验(Linux命令: md5sum, sha1sum)并将结果与该网站列出的值进行比对, 如有不符, 必须重新下载

使用 USB 连接线将 Kindle 连接至电脑, 打开 Kindle 存储分区, 将升级文件复制到 Kindle 根目录中, 断开与电脑的连接, 此时, “更新您的 Kindle”按钮已经可用, 选择更新, Kindle 将重启一次完成升级

安装自定义字体:

打开http://font.chinaz.com/{: target=”_blank” }(我用过的), 购买字体下载, Kindle 支持的字体扩展名有: .ttf, .ttc 或 .otf, 将 Kindle 与电脑连接, 把下载好的字体文件(可能需要解压, 直到得到 .ttf, .ttc 或 .otf 文件)复制到 Kindle 根目录下的 fonts 文件夹, 断开与电脑的连接, 在 Kindle 打开一本书, 即可选取新安装的字体使用.
