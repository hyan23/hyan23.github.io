Chrome插件

我所有插件都在这里了…

  • AdBlock - 能有效屏蔽网页中的广告, 建议用来对付流氓网站
  • 哔哩哔哩助手 - 可以推送关注UP主新视频提醒, 自动领瓜子等
  • Chrono Download Manager - 一个下载管理器
  • crxMouse Chrome™ Gestures - 为浏览器增加手势功能
  • Grammarly for Chrome - Grammarly语法检查, 检查你在富文本框里输入的英语拼写和语法并提出修正建议
  • Kami - PDF and Document Markup - 功能很全的PDF阅读和编辑器, 在线保存批注
  • Noisli - 提高专注度和生产率
  • Personal Blocklist (by Google) - 屏蔽你不希望看到的网站出现在Google搜索结果中
  • Proxy SwitchyOmega - Proxy SwitchyOmega
  • Search by Image (by Google) - 为图片右键菜单添加Google”以图搜图”按钮
  • Send to Kindle (by Klip.me) - 将网页发送到你的Kindle阅读器, 没用过
  • StayFocusd - 限制一天内你浏览特定网站的时间(如b站)
  • Syntaxtic! - 为文本文档(代码)显示语法高亮
  • Tampermonkey - 没用过
  • TransOver - 多语言鼠标悬浮翻译, 看英文(外文)网页利器
  • Turn Off the Lights - 关灯, 暗淡视频区域外的区域
  • Video Downloader professional - 下载网页中的视频
  • WOT Web of Trust, Website Reputation Ratings - 查看基于用户体验给出的任何网站的声誉和安全信息
  • 百度广告屏蔽 - 屏蔽百度搜索结果中的广告
  • 眼不见心不烦（新浪微博） - 屏蔽新浪微博推广、广告等内容

工具

它们很多都可以直接使用 apt-get 来安装

  • 输入法框架 - ibus

我们知道常见的输入法框架有两个: ibus 和 fcitx, 从我自己的使用境况来看, ibus 更加稳定

  • 输入法 - 中州韵

完全可配置的中文输入法, 使用 yaml 语言来定制

  • 虚拟机 - VirtualBox

Oracle 公司开发的虚拟机软件, 是开源和免费的

  • Dos 模拟器 - DOS Box

  • 邮件客户端 - Icedove

现在配置 gmail 有点问题, 记住 Authentication Method 要选择 Normal password

  • 视频播放器 - MPlayer

最好的视频播放器

  • 远程演示 - Teamviewer

最好的远程桌面解决方案

  • 编辑器 - Code

由微软开发的通用代码编辑器, 在 Atom 的基础上开发, 内置终端, 可调试, 支持插件, 自定义快捷键

  • 博客生成器 - Jekyll

一款轻量级的静态网站生成器, 把网站的源码 push 到 Github Pages, 你就拥有了自己的博客

  • 查单词 - 有道词典

Linux 有许多优秀的字典软件, 如: GoldenDict, StarDict 等, 但有道词典是其中唯一支持屏幕取词功能的, 如果你的电脑屏幕是FHD(全高清)的, 如果觉得字体太小, 可以下载我的补丁来改善它 链接

  • 办公 - LibreOffice

  • SSH - PuTTY

  • 玩GBA游戏 - visualboyadvance

  • 玩NES游戏 - higan

  • 程序有没有内存泄漏? - valgrind

知道程序泄漏了内存是很重要的, 这个工具可以帮你检查哪些分配的内存没有释放

  • 看chm文档 - chmsee

  • 十六进制编辑 - bless

  • 基于图像识别的自动化操作 - sikuli

可以用 python 或 Java 编程, 它有一个 IDE(python), 提供截图和截区域(Region)工具, 可视化地调节相似
