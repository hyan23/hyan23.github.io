Projects

Project Name Time Span Description Screenshot(s)
tree 2016.04 - 2016.08 A toy multi-tasking, GUI Operating System built from scratch, it is written purely in 32-bit Assembly language and targeted Intel IA-32 architecture. It provides memory allocation / de-allocation, dynamic linking, system calls, a graphics library, task management, window management / event mechanism, a keyboard driver, and a mouse driver. Executables are extension’d with .bin, and can be authored and compiled on other system which runs NASM
cnes 2017.10 - 2018.06 A Nintendo NES/FC emulator to support my undergraduate thesis, built upon C and SDL1.2. Theoretically, 80% of the original cartridge games can be emulated and played with this emulator. In addition, I implemented a bunch of advanced features that commercial emulators commonly have, e.g., snapshot, USB gamestick support, stereo audio, APU visualization, key mappings. Other features were planned, include speed control, cheating, post processing, screenshot, game library (GUI), crossing platforms
8086asm 2016.10 - Now An essential Assembly function library targeted 8086 processor, inspired by Ivrine32 Library, consists of modules like I/O, strings, arithmetic, files, system, utility, macro library. User can use more than 90 reliable, concise functions to develop 16-bit Assembly application conveniently. User’s manual is available here
TypeScript 中文手册 2019.11 - Now Chinese translation of the TypeScript Handbook, they actually provide official Chinese version, I translated it as a practice, my version consists of 75k Chinese characters
