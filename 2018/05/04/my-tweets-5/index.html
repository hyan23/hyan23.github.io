关于在 4 月 6 号 tweet 中提到的百度首页推送一事, 现在在 “即刻” APP 找到了解决方案, 在手机端打开 https://baidu.com/?tn=simple 即可访问清爽版的百度搜索主页.
2.
师兄帮帮, 是一款大学生知识付费分享平台, 对那些在某科学得好的同学, 可以在该 APP 帮助解答学术问题获得 “奖学金”, 对于学习有困难的同学, 可以在该 APP 付费得到帮助. 我试用了一次, 我遇到的这位学长解答问题快, 责任心强. 该 APP 设置的金额合理(默认为 3 - 50 元不等), 会根据你设置的金额多少推送给一定数量擅长该问题的同学. 基本上, 在学习上遇上的问题都能在这个 APP 得到解决, 如果你觉得理解数学上的某些过程有困难, 又不想过多地麻烦自己的好朋友, 不妨试试这款 APP.

很多单车的车把都可以任意旋转, 比如说 ofo, 通常, 车把旋转 360 度以后, 制动功能就会受到影响, 因为刹车线绕车把扭曲, 摩擦力增大了, 所以不容易使用手柄进行牵引. 对于共享单车来说, 如果不在发车前检查, 骑行就会有很大的安全隐患, 而解决方案就是限制车把的旋转角度. 我本来想找时间把这条建议反馈给 ofo 的, 昨天, 摩拜单车免押金了, 我体验了一下摩拜单车, 远优于 ofo 的骑行和使用体验让我怀疑 ofo 研发部到底有没有在用心做产品, 所以就不想把这条建议反馈给官方了, 但当我写下来, 又改变了主意, 事关安全的建议憋在心里会让我不安的…那我再找时间反馈.

2.
中国的乘车环境真的跟菜市场似的, 打电话、看视频、聊天、小孩哭闹. 不管别人怎么做, 我现在要更严格要求自己, 除了从来不使用手机外放, 现在要注意不在车上聊天(只说必要的话), 尽量不接电话. 对于小孩, 我是真的不理解现在的小孩为什么那么没教养, 我小的时候跟他们完全不同, 印象中, 我在外面从来不敢大声讲话, 更别说大哭大闹, 调皮捣蛋了, 今天在地铁上还看到一个小孩儿疯狂扇他妈妈耳光, 吸引了很多人的注意力, 他妈妈却只是眼神制止. 父母过度溺爱, 教子无方怎么能从小树立起孩子正确的世界观呢, 这是做父母的不负责任. 我认为, 小孩子5岁以前所受的教育非常重要, 不说教他诚实、勤奋、先人后己这些大道理, 起码要让孩子对世界、对父母、对老师抱有敬畏之心, 这是大了一些后能够顺利接受正统教育的基础.
3.
前几天给我的 IPhone 安装了百度输入法, 这是我第一次在手机上安装第三方输入法软件. 过去我的安卓系统安装的是百度输入法, 而这么久依然没有在 IPhone 上安装是因为我对国外产品、技术的过度迷信, 以至于我的大脑进行了持续两年的自我欺骗, 事实是, IPhone 自带中文输入法除了 3DTouch 移动光标的体验相对(手指光标定位不好用)人性化以外, 输入功能完全比不上国产中文输入法, 而且在 4.7 寸屏上的全键盘虚拟键盘误按率极高, 以至于完整无错误地输入一句话成了小概率事件.

2018.06.08

1.
更正错别字: 今天介绍更正错别字的方法, 如果我们在聊天中打错了一个字, 很多人会重新打一遍正确的字发过去, 在国外, 人们会在这个字前面加上 ‘‘ 符号表明这个字是对前面那句话其中一个字的修正, 这种表示方法使错误的更正更具指向性.
2.
*eclipse动态加载: 我一直把 eclipse 作为我的首选 Java IDE, 直到最近, 我才发现 eclipse 有字节码文件动态更新的功能(当然这肯定是 jvm 本来就有的特性), 目前的经验是, 只要我们不对项目中某个 class 的结构进行修改(新增、修改成员变量/方法), 而只修改了代码逻辑部分, 保存文件后, eclipse 会自动编译已过期的 class, 并且 patch 到当前调试环境中, 使我们可以直接调试新的代码(而不用重新打开调试, 输入一遍数据), 从而在一定程度上提升调试效率, 本来这个功能也是我在看 Java 入门书就提到过的(动态语言/脚本语言的特点), 但最近才体验到.
